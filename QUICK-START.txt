   → Run installer and restart computer
   → Launch Docker Desktop (wait for green whale icon)

TO START:
  1. Make sure Docker Desktop is running (green whale icon)
  2. Double-click: START-HEADOFFICE.bat
  3. Wait 60 seconds - browser opens automatically
  4. Application ready at: http://localhost:3000

TO STOP:
  1. Double-click: STOP-HEADOFFICE.bat
  2. Wait for confirmation message
  3. Done!
